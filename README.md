# EquipWise
## NCI-BSHC-Year2-DSA-CA1-x24218481

**### Overview**
A pilot application for buying & selling **second-hand sports equipement**. Developed by a student at _National College of Ireland_ for a Countinous Assesment. Uses new knowledge obtained from **Data Structures & Algorithms** as well previous knowledge from **Graphical User Interfaces** learnt at _Colaiste Dhúlaigh CFE_.
### ADTs:
SLL (Binary Search Tree) - Stores / updates items
Queue - Stores orders & deletes items
BST (Binary Search Tree) - Performs searches
### CRUD:
Create - Users can create new items by adding to the SLL & new orders by adding to the Q
Read - Users can view items from the SLL
Update - Users can update the SLL by adding items & update the Q by adding orders
Delete - Items are deleted by the user once an order is placed

Open the project on Apache NetBeans, then locate and run the main class (MainFrame.java). The application should launch on the BrowseSearchUI.java screen.

_**Note:** Receipts are saved in the project directory (EquipWisePilotApp\EquipWise)... Interface might be slow, especially when searching for items or switching between panels!_
